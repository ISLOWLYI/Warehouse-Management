# Warehouse Management

Веб-приложение для управления складскими запасами. Позволяет вести учет товаров, контролировать их количество, местоположение и статус.

## Стек технологий

- **Backend:** Ruby on Rails 8.1, PostgreSQL
- **Frontend:** HTML + Tailwind CSS (CDN)
- **Аутентификация:** Devise
- **Контейнеризация:** Docker + docker-compose

## Быстрый старт

### 1. Запуск через Docker

```bash
docker compose up --build
```

Приложение будет доступно по адресу: [http://localhost:3000](http://localhost:3000)

### 2. Накатить миграции и создать демо-данные

В отдельном терминале выполните:

```bash
docker compose exec web bundle exec rails db:migrate
docker compose exec web bundle exec rails db:seed
```

### 3. Вход в систему

После выполнения `db:seed` создается аккаунт администратора:

| Поле | Значение |
|------|----------|
| Email | `admin@warehouse.ru` |
| Пароль | `password123` |

Откройте [http://localhost:3000/users/sign_in](http://localhost:3000/users/sign_in) и войдите с указанными данными.

> **Важно:** Регистрация новых пользователей доступна **только по приглашению администратора**. Чтобы пригласить пользователя, зайдите под админом → выберите склад → раздел **Участники** → введите email и роль → нажмите **Пригласить**.

## Основные возможности

- Управление складами (создание, редактирование, удаление)
- Учет товаров с артикулом, категорией, количеством и местоположением
- Визуальная карта склада с подсветкой ячейки товара
- Ролевая модель: Администратор, Работник, Поставщик
- Приглашение участников по email
- Сортировка и фильтрация товаров
- Адаптивный интерфейс (десктоп + мобильные устройства)

## Полезные команды

### Консоль Rails

```bash
docker compose exec web bin/rails console
```

### Миграции базы данных

```bash
docker compose exec web bin/rails db:migrate
```

### Создание демо-данных (seeds)

```bash
docker compose exec web bin/rails db:seed
```

### Полный сброс базы данных

```bash
docker compose exec web bin/rails db:drop db:create db:migrate db:seed
```
